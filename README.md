Customer Churn Prediction

Problem Statement
Predict whether a telecom customer will churn based on account, usage, and service features.

Tools and Technologies
Python
Pandas, NumPy
Scikit-learn
Imbalanced-learn (SMOTE)
Random Forest, Logistic Regression

Methodology
Performed data cleaning and preprocessing
Encoded categorical variables
Handled class imbalance using SMOTE
Trained Logistic Regression and Random Forest models
Evaluated models using ROC-AUC, F1-score, and confusion matrix
Analyzed feature importance to identify key churn drivers

Results
Achieved ROC-AUC of 0.822 using Random Forest

Business Impact
This model helps telecom companies identify high-risk customers early and take retention actions to reduce revenue loss.
